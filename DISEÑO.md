\# Diseño de la API – Miguel Ángel Fernández Sánchez



\## ¿Qué es la API?

La API es el backend de GfitApp. Se encarga de recibir

peticiones del frontend y devolver datos del gimnasio.



\## Endpoints que voy a crear:



\### Clases

\- GET /clases → devuelve la lista de clases disponibles

\- POST /clases → crea una clase nueva



\### Socios

\- GET /socios → devuelve la lista de socios

\- POST /socios → registra un socio nuevo



\### Reservas

\- GET /reservas → devuelve todas las reservas

\- POST /reservas → crea una reserva nueva



\## Tecnología

\- Python + Flask

\- Los datos los proporciona la base de datos de Ainhoa
