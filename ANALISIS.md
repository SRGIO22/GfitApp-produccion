## Introducción

Tras examinar el trabajo realizado por el grupo de prácticas 1, considero que para la implementación del sistema de login será necesario llevar a cabo las siguientes tareas:

## Tareas necesarias

### 1. Modificación de la base de datos

Añadir las tablas `ADMINISTRADORES` y `CONTRASEÑAS` a la base de datos para gestionar la autenticación y los distintos tipos de usuarios.

### 2. Actualización de la interfaz de usuario

Incorporar un apartado de inicio de sesión en la interfaz de usuario que permita el acceso mediante credenciales.

### 3. Implementación del sistema de autenticación y roles

Desarrollar el sistema de login incluyendo:

- Gestión de autenticación de usuarios.
- Restricciones de acceso según el rol.
- Privilegios diferenciados para administradores y socios.
- Control de permisos en las distintas funcionalidades de la aplicación.
