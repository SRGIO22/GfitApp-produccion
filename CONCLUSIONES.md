\# Conclusiones – Miguel Ángel Fernández Sánchez



\## Reflexión personal



Al empezar las prácticas no tenía ningún conocimiento de Python,

Flask, APIs ni Git. Todo era nuevo para mí.



He aprendido qué es el backend y cómo funciona, cómo crear una

API REST con Flask, cómo conectarla a una base de datos real,

y cómo usar Git y GitHub para gestionar mi trabajo.



Lo más difícil ha sido arrancar desde cero sin haber visto estos

temas en clase. Lo más satisfactorio ha sido ver la API funcionando

con datos reales en el navegador.



Ha sido una experiencia muy intensa pero muy útil para entender

cómo funciona el desarrollo de software en un entorno real.
